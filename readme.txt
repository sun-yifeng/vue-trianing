第15节课相关说明

1、安装node
node -v

2、安装cnpm
npm install -g cnpm --registry=https://registry.npm.taobao.org

3、安装vue-cli
cnpm install -g @vue/cli

4、安装webpack
cnpm install -g webpack

5、启动图形界面
vue ui
